# Skyline Travel Booking

A modern frontend web application for a full-service travel agency, handling airline transportation and hotel reservations. Built with Vue 3, Vite, and Tailwind CSS.
</div>

---

## 🚀 Features

- 📱 Responsive Design for mobile, tablet, and desktop
- 🧭 Seamless page navigation using Vue Router
- 🗂️ State management with Pinia
- 🎨 Modern UI with Tailwind CSS
- 🔔 Interactive toast notifications using Sonner
- ✉️ Contact form interface with instant feedback
- 📰 Multi-page layout including Home, About, News, and Contact pages

---

## 🛠️ Tech Stack

- **Framework:** Vue 3 (Composition API)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Routing:** Vue Router
- **State Management:** Pinia
- **Icons:** Lucide Vue Next
- **Notifications:** Vue Sonner

## 📦 Project Setup

### Prerequisites

Make sure you have the following installed:

- Node.js v20.19.0 or higher
- npm

Check versions:

```bash
node -v
npm -v
```

---

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/INT250-GA1-HiewKhaoHorYing-The-Travel-Booking.git
```

Navigate to the project folder:

```bash
cd INT250-GA1-HiewKhaoHorYing-The-Travel-Booking
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
